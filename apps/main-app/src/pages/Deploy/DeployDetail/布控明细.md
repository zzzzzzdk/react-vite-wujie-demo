## 布控明细

### 操作按钮

| 按钮     | 显示条件                                                         |
| -------- | ---------------------------------------------------------------- |
| 查看     | 无条件                                                           |
| 编辑     | 当前用户是创建人&&布控单未关闭                                   |
| 撤销     | 当前用户是创建人&&待审批                                         |
| 审批     | 当前用户是审批人或所在部门管理&&当前用户有审批权限&&布控单待审批 |
| 关闭     | 当前用户是创建人&&布控单为布控中                                 |
| 重新布控 | 当前用户是创建人&&布控单为已关闭                                 |
